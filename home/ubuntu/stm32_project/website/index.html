STM32F407 HID to Serial/CAN Project

A versatile input-to-output mapping system for the STM32F407 microcontroller

Project Overview

This project implements a versatile input-to-output mapping system for the STM32F407 microcontroller, specifically designed for the STM32mega development board. The system allows multiple HID input devices (keyboards, mice, gamepads, etc.) to be connected to the STM32F407, with their inputs mapped to either serial or CAN bus outputs.

Key Features

Multiple HID Device Support

Connect and use various USB HID devices simultaneously

Flexible Mapping System

Map any input to any output with configurable parameters

Dual Output Options

Send mapped inputs to Serial UART or CAN bus

1.28" Round TFT Display

Visual feedback and status information via GC9A01 controller
